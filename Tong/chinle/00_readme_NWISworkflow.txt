##NWIS workflow

1. Look up NWIS local aquifer codes for the following units: Chinle group [COMPLETE]
2. Download site descriptions for each unit with >0 well [COMPLETE]
3. Filter each unit to sites with >=1 water quality record
4. Compile all Site Numbers with available water quality records
5. Move over to https://waterdata.usgs.gov/download-samples to download data with list of Site Numbers (limit of IDs per search is ~150)
6. Download data for results, full physical chemical
7. Compile all results for each unit group
8. Explore data for suitability with model
